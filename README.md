## ข้อมูลทั่วไป

โปรแกรมเล็กๆ ใช้สำหรับแก้ปัญหาที่ใครเขาไม่เป็นกัน สร้างด้วยภาษา [rust](https://rust-lang.org/) โดยใช้ไลบรารี่ [eframe](https://docs.rs/eframe/latest/eframe/) ในระบบ GUI และใช้ [windows](https://docs.rs/windows/latest/windows/) crate ดึงข้อมูล RAM สุดท้ายก็ใช้ [winres](https://docs.rs/winres/latest/winres/) ในการช่วย build พวกไอค่อน.
